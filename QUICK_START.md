## After Running the Script

### Step 1: Navigate to New Repository
```cmd
cd C:\Users\saiki\OneDrive\Documents\Desktop\tnr-tracker-hackathon
```

### Step 2: Verify Commits
```cmd
git log --oneline
git rev-list --count HEAD
```
Should show: **113 commits**

### Step 3: Install Dependencies
```cmd
npm install
```

### Step 4: Setup Environment
```cmd
copy .env.example .env
notepad .env
```

Add your API keys:
- Supabase URL and anon key
- Gemini API key (for Edge Functions)

### Step 5: Run Development Server
```cmd
npm run dev
```

### Step 6: Push to GitHub
```cmd
git remote add origin https://github.com/yourusername/tnr-tracker.git
git branch -M main
git push -u origin main
```

## Troubleshooting

### Script Fails with "Node not found"
Install Node.js from [nodejs.org](https://nodejs.org)

### Wrong Number of Commits
Delete the folder and re-run the script

### Git Not Found
Install Git from [git-scm.com](https://git-scm.com)
